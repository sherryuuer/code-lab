# code-lab

個人用の Coding Practice Workspace です。Language 別の練習と、Code Review 用の練習領域に分けています。

## Structure

- `java/`: Java の Practice Project と Language Exercise。
- `javascript/`: JavaScript、DOM、Async、Browser 関連の Exercise。
- `python/`: Python の Language Basics、Automation、Testing、Data Processing、Utility 系の Exercise。
- `SQL/`: SQL Notes と Database Command Reference。
- `code-review/`: Code Review 専用の Practice Area。

## Code Review 練習

Data Architecture 観点の Code Review 練習には、`code-review/data-architecture/` を使います。

現在の重点領域：

- SQL Review
- Python Data Pipeline Review
